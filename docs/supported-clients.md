# Supported Clients

## Claude Desktop

- macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
- Windows: `%APPDATA%/Claude/claude_desktop_config.json`
- Linux: `~/.config/Claude/claude_desktop_config.json`

## VS Code

- `.vscode/mcp.json`
- `.mcp.json`
- Best-effort user settings detection

## Cursor

- `.cursor/mcp.json`
- `.cursor/settings.json`
- `.mcp.json`

## Codex

- `.mcp.json`
- `mcp.json`
- `.codex/mcp.json`
- `.codex/config.json`

Codex support is best effort because config paths may vary.

## Gemini CLI

- `.gemini/mcp.json`
- `.gemini/settings.json`
- `.mcp.json`

Gemini support is best effort because config paths may vary.
